# MPS-FOC 芯片应用手册

## 1. 项目评审结论

当前项目已经形成完整的“电源链路 + 驱动链路 + 角度反馈链路”：`48V -> 12V -> 5V -> 3.3V` 分级供电稳定，`MP6539B` 三相驱动链路可正常出 PWM，`MA600A` 角度反馈已接入 FOC 主环，速度环和位置环可运行，电流环参数也已经在代码中按带宽法计算并初始化。对比赛场景来说，这套方案的优势是启动快、可复现性高、展示稳定。

---

## 2. 本项目中的 MPS 芯片分工

1. `MA600A`：电机角度反馈（FOC 位置基准）
2. `MP6539B`：三相栅极驱动（驱动 6 颗外置 MOS）
3. `MP4583`：`48V -> 12V` 高压 Buck
4. `MPM3632S`：`12V -> 5V` Buck 电源模块
5. `MP20051`：`5V -> 3.3V` 低噪声 LDO
6. `MIE1W0505`：隔离 `5V -> ISO_5V`（通信隔离侧）

---

## 3. 分芯片应用说明（含数据与项目落地）

### 3.1 MA600A（位置传感）

MA600A 是本项目 FOC 的角度基准，固件在 `20kHz` 中断里通过软 SPI 读角度（`MA600_ReadRaw_Fast()`），并用 `theta = raw * (2π/65536)` 转机械角，当前实现的角度量化步进是 `0.00009587 rad`（`0.00549°`）；再乘电机 `14` 极对得到电角度，等效电角分辨率 `0.0769°`。这个分辨率级别在低速下能明显减少 Park 变换输入抖动，速度环/位置环会更平顺，尤其是小角度定位时更容易“贴着目标走”。

### 3.2 MP6539B（3 相栅极驱动）

驱动芯片承担“把 MCU 的控制量变成三相功率级开关动作”的核心角色，项目中由 EPWM 输出 `6` 路互补 PWM（A/B/C 三相上下桥）送入 MP6539B，再去驱动外置 `6` 颗 BSC030N08NS5；按器件数据（同系列口径）可覆盖 `8V~100V` 驱动电压范围，栅极驱动能力约 `IOH=800mA`、`IOL=1A`、`6` 路驱动通道。固件里 `nSLEEP` 由 GPIO 直接控制，上电后先使能再放 PWM，且 MCU 侧已配置死区：`deadband=100` 计数（`100MHz` TBCLK 下约 `1us`），这套配置对比赛调试非常友好，既能快速跑起来，也有明确保护边界。

### 3.3 MP4583（48V -> 12V 高压 Buck）

MP4583 把母线 `48V` 先降到 `12V`，直接把高压与控制/接口电源域隔开；发布版原理图参数是 `R6=180k`、`R5=12.58k`、`L6=22uH`，按常见 FB 公式 `Vout = VFB * (1 + Rtop/Rbot)`、`VFB≈0.8V` 计算：`Vout ≈ 0.8*(1+180/12.58)=12.25V`，与目标 12V 一致。把最高压这一级先处理掉后，后级 5V/3.3V 电源设计压力会明显降低，比赛现场电池波动时系统也更稳。

### 3.4 MPM3632S（12V -> 5V 电源模块）

MPM3632S 是中间母线电源，承担全板 `5V` 供电主力；器件本身是集成电感降压模块，典型能力 `4V~18V` 输入、`3A` 输出、`2.2MHz` 开关频率。项目原理图给出的分压是 `R35=43k`、`R36=8.2k`，按 `VFB≈0.8V` 计算得到 `Vout ≈ 4.99V`，与 5V 目标精确对齐。这个模块把外围设计复杂度压低了，硬件迭代时间更短，适合比赛周期内快速出稳定板。

### 3.5 MP20051（5V -> 3.3V LDO）

MP20051 是控制侧“末级净化电源”，给 MCU 与逻辑域提供更干净的 3.3V；器件参数层面有 `1A` 输出能力、`PSRR=63dB@1kHz`、`Dropout=140mV@1A`、噪声约 `13uVrms`，在 FOC 的 ADC 采样与角度处理场景里很实用。原理图分压为 `R42=75k`、`R41=24k`，按 `Vout = 0.8*(1+75/24)` 可得约 `3.3V`。此外 LDO 热设计可直接按 `P=(Vin-Vout)*Iout` 估算，例如 `5V->3.3V`、`0.2A` 时约 `0.34W`，比赛中按这个量级预留散热即可。

### 3.6 MIE1W0505（隔离电源）

MIE1W0505 给 CAN 隔离侧提供独立 `ISO_5V`，把功率地噪声和通信地做物理隔离；从型号可直接看出是 `5V->5V`、`1W` 隔离电源，折算输出电流上限约 `Iout_max = 1W / 5V = 0.2A`。这个隔离电源在电机大电流开关噪声环境下非常关键，通信链路不容易被功率回流“带脏”，长时间演示时总线稳定性明显更好。

## 4. 开关电源参数怎么计算（本项目可直接套用）

### 4.1 输出电压分压

```text
Vout = VFB * (1 + Rtop / Rbot)
```

1. `MP4583`（48V->12V）：`Rtop=180k, Rbot=12.58k`，`Vout≈12.25V`
2. `MPM3632S`（12V->5V）：`Rtop=43k, Rbot=8.2k`，`Vout≈4.99V`

### 4.2 占空比一阶估算

```text
D ≈ Vout / Vin
```

1. `48V -> 12V`：`D≈0.25`
2. `12V -> 5V`：`D≈0.417`

### 4.3 电感纹波电流估算（Buck）

```text
ΔIL = (Vin - Vout) * D / (L * fsw)
```

以 MP4583 级举例：若 `Vin=48V, Vout=12V, D=0.25, L=22uH`，假设 `fsw=500kHz`，则 `ΔIL≈0.82A`。这个值可用于反推电感和输出电容选型。

### 4.4 输出电容估算（纹波约束）

```text
Cout >= ΔIL / (8 * fsw * ΔVout)
```

延续上例，若目标纹波 `ΔVout=50mV`，则 `Cout>=4.1uF`（工程上通常会留余量并并联低 ESR 电容）。

### 4.5 LDO 功耗估算（MP20051）

```text
P_LDO = (Vin - Vout) * Iout
```

`5V -> 3.3V`、`Iout=0.2A` 时约 `0.34W`，可作为热设计和封装温升评估输入。

## 5. 代码中的关键实参（和芯片联动）

1. 控制频率：`20kHz`（`PWM_PERIOD=2500`，中心对齐）
2. 电流环带宽：`1000Hz`
3. 电流环 PI（按代码电机参数计算）：`Kp≈2.586`，`Ki≈4555`
4. 速度计算常数：`190985.9`（`20kHz * 60 / 2π`）
5. 总线电压换算：`Vbus = 0.01933594 * ADC`

这些值已经在工程中闭环使用，手册中的芯片参数计算与控制参数是同一套口径。
